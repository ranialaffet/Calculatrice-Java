Démonstration du controleur de la calculatrice

http://ip_serveur:port/calculatrice_backend/calculate?operation=xx&operande1=yy&operande2=zz

ip_serveur : adresse IP du serveur

port : port d'écoute du serveur

xx : add, sub, mul ou div

yy : valeur de l'opérande 1

zz : valeur de l'opérande 2

Exemple:

http://localhost:8080/calculatrice_backend/calculate?operande1=10&operande2=5&operation=add
